## Bahon – Smart Transportation Mobile App

**Bahon** is an Android app developed for smart public transportation. The app is built with **Java** and **XML**, using **Firebase Authentication, Firestore, Realtime Database, and Cloud Functions** for backend operations.

### Main Features

* **User Registration & Login**
* **QR Code generation** to start and end journeys
* **Real-time Tracking** of the user’s current journey location
* **Travel History** with distance, time, and fare details
* **Balance Check and Recharge** with recharge history
* **Fare Calculator** for estimating journey costs
* **Profile Settings** for managing user details
* **Supervisor Interface** to monitor passengers and payments
